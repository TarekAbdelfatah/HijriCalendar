Hijri Calendar - JavaScript API

مكتبة JavaScript نقية - بدون Angular أو jQuery

Available Functions

createDatePicker(options)
إنشاء date picker مع popup التقويم
hijriToGregorianStr(str)
"1447/10/22" → "2026/04/08"
gregorianToHijriStr(str)
"2026/04/08" → "1447/10/22"
todayHijri()
التاريخ الهجري الحالي
todayGregorian()
التاريخ الميلادي الحالي

1. Installation

<script type="module">
  import { createDatePicker, hijriToGregorianStr } from './hijri-calendar.lib.js';
</script>

2. Basic Usage

const picker = createDatePicker({
  container: '#myInput',
  onSelect: (dates) => {
    console.log('Hijri:', dates.hijri);   // "1447/10/22"
    console.log('Greg:', dates.greg);    // "2026/04/08"
    return null;
  }
});

// Convert
hijriToGregorianStr("1447/10/22"); // "2026/04/08"
gregorianToHijriStr("2026/04/08"); // "1447/10/22"

3. Options

OptionTypeDescription
containerstring | HTMLElementInput element or selector
defaultView'hijri' | 'gregorian'Initial view (default: 'hijri')
outputFormatstring'yyyy/mm/dd', 'dd/mm/yyyy', 'yyyy-mm-dd'
onSelectfunctionCallback when date is selected

4. Validation Example

createDatePicker({
  container: '#input',
  onSelect: (dates) => {
    if (dates.hijri < '1447/01/01') {
      return 'Date must be after 1447/01/01';
    }
    return null; // success
  }
});

5. Auto-Normalize

المكتبة تطبيع تلقائياً أي تنسيق:

hijriToGregorianStr("2026/04/08");  // yyyy/mm/dd ✓
hijriToGregorianStr("08/04/2026");  // dd/mm/yyyy ✓
hijriToGregorianStr("08-04-2026");  // dd-mm-yyyy ✓
