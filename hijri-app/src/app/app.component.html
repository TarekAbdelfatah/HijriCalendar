📚 JavaScript API - Standalone Library

دليل شامل لاستخدام مكتبة JavaScript النقية (بدون Angular أو jQuery)

1️⃣ التثبيت

انسخ ملفين إلى مشروعك:

src/app/hijri-calendar/
├── hijri-calendar.lib.ts      ← المكتبة
└── hijri-calender.directive.ts ← الدايريكتيف
2️⃣ الاستخدام الأساسي

أضف hijri-calender على أي input:

// app.component.ts
import { HijriCalenderDirective } from './hijri-calendar/hijri-calender.directive';

@Component({
  imports: [FormsModule, HijriCalenderDirective],
  ...
})
export class AppComponent { }
<!-- template.html -->
<input hijri-calender [(ngModel)]="model.date" />
3️⃣ bindValue - التحكم في صيغة القيمة

حدد هل ngModel يستقبل هجري أو ميلادي:

<!-- افتراضي: ngModel = هجري -->
<input hijri-calender [(ngModel)]="model.hijriDate" />

<!-- ngModel = ميلادي -->
<input hijri-calender [bindValue]="'gregorian'" [(ngModel)]="model.gregDate" />
النتيجة: كلاهما يعرض popup هجري، الفرق فقط في القيمة المُخرَّجة
4️⃣ التحقق من التاريخ

استخدم onSelect callback للتحقق:

// app.component.ts
onSelect(dates: { hijri: string; greg: string }) {
  if (dates.hijri < '1447/01/01') {
    return { isValid: false, errorMessage: 'التاريخ قبل المسموح' };
  }
  return { isValid: true };
}
<input hijri-calender 
       (onSelect)="onSelect($event)"
       [(ngModel)]="model.date" />
