دليل التشغيل السريع لمكتبة CoreComponents

🚀 التشغيل السريع

1
افتح المجلد في VSCode
2
افتح الطرفية: Ctrl + ~
3
شغّل الأمر: npm run dev
4

npm run build بناء للمشروع
npm run preview معاينة البناء
